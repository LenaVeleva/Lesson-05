Виды базовых селекторов:

  1. Селекторы по тегу - предназначен для выбора элементов по имени тега. Создается внутри тега head, пишется тег {свойство: значение;} По приоритетности ниже чем селектор по id и class
  2. Селекторы по классу - предназначен для выбора элементов по классу (исп class). Создается внутри тега style в head таким образом: .название класса {свойство: значение;}. А в body в теге пишется class="название класса" По приоритетности ниже чем селектор по id, но есть преимущество, что название можно повторять несколько раз на странице
  3. Селекторы по идентификатору (id) - предназначен для выбора элемента по идентификатору (исп id). Создается внутри тега style в head таким образом: #название id{свойство: значение;} А в body в теге пишется id="название id". Высокая приоритетности, но не может повторятся дважды!!!
  4. Селекторы по по нескольким классам -предназначен для выбора элементов по нескольким классам, удовлетворяющие одновременно сразу нескольким условиям. Создается внутри тега style в head таким образом: .название класса1.название класса2 {свойство: значение;}. А в body в теге пишется class="название класса1"
  5. Селекторы по вложенности. Создается внутри тега style в head таким образом: .название класса1 .название класса2 {свойство: значение;}. А в body в теге пишется class="название класса1, также вложен второй тег, у которого также есть class="название класса2 "
  6. Уриверсальный селектор (через *) - одновременно один стиль для всех элементов. Показала в файле new.html
